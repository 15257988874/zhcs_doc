表单使用方法
1.表单初始化(返回表单对象)
2.表单操作(初始化或者赋值操作)
3.弹出表单
注: 假设表单初始化对象为popup(ycya.ui.autoform(para)的返回结果)
1.表单初始化(ycya.ui.autoform())
用于初始化表单弹窗(重点)
参数 数据类型 示例 说明
id String 'yui-add' 表单容器ID必填
filter String 'yui-add' 表单过滤器，用于检测表单提交,触发验证(唯一性)
注: 除高级查询以及查看弹窗生成外,必传
isAdvanced Boolean false 是否高级查询,默认false
allowEmpty Boolean false 验证通过后,返回的数据中是否允许有空值,默认false
colWidth String 'two' 一行2列或一行1列，支持(one,two),若要扩展,则需要扩展ycyaAq.less
verType String 'tips' 表单验证失败提示方式,该参数有3个值,msg tips alert,默认tips
submitFn Function submitFn: function (data) { console.log(data); } 该函数返回一个实参,即表单验证通过后的数据
callback Function callback:function(obj){ console.log(obj); } 该函数返回一个实参,即当前表单对象,用于处理表单初始化完成后的额外操作
item Array 详见表单字段 表单字段定义，格式为[{},{}],必填
pcfg Object 详见表单图片字段 用于上传图片
map Object 详见表单地图字段 用于创建左侧表单信息，右侧地图的表单
item-表单字段
参数 数据类型 示例 说明
label String '开始时间' 字段标签名,必填
name String 'startTime' 字段名,必填
elmType String 'select' 元素类型,支持(input,textarea,select,p),默认input
isBlock Boolean true 该字段是否独占一行
inputType String 'password' 生成密码框,暂只支持(password),自动MD5加密传输
verify String 'required||phone ' layui内置验证,支持多条件验证(中间'||'隔开),可自定义验证
qt String 'gt' 用于高级查询类,支持(eq,gt,lt,like,in)
cfg Object time:{type: 'datetime',format:'yyyy-MM-dd HH:mm:ss'}
select:{
count:4,
search:true,
url:'select.js',
cfg:{
data:{'1':'男','2':'女'},
type:"get",
searchName:'foodName',//自定义搜索内容的key值
keyName: 'foodName', //自定义返回数据中name的key, 默认 name
keyVal: 'id',  //自定义返回数据中value的key, 默认 value
beforeSuccess:function(id, url, searchVal, result){
    return result.data.data.data;
}
}
}
用于时间弹窗以及select的配置( select支持本地和服务器加载，请参照示例,本地加载不传url和配置cfg中的各项自定义参数,优先服务器加载 )
count: 0代表不限制选择个数 1代表单选 大于1代表最多选择几个
search:是否开启搜索
data:本地加载数据 
type:请求方式,默认post
searchName:'',//自定义搜索内容的key值
keyName: '', //自定义返回数据中name的key, 默认 name
keyVal: '',  //自定义返回数据中value的key, 默认 value
beforeSuccess:function(id, url, searchVal, result){
    //处理数据格式
    return result.data.data.data;
}
dt String 'date' 支持(data,switch,text)
handle Function 无 表单元素事件函数,用于扩展
eventName String 'click' 表单元素事件名称,该参数包含2个大类
1.jquery支持的事件名称 2.内置事件名称,暂支持(tree,address)
treeCfg Object { url:'/page/default/tpl/tree.html?p=tree', title:'请选择部门', name:'name' } 树类弹窗,该参数配置遵循ztree树配置
pcfg-图片上传
参数 数据类型 示例 说明
btns Boolean 是否需要按钮组 按钮组,默认true
auto Boolean 是否自动上传 默认true
url String app.url+ 'file/up' 图片上传地址
max Number 3 最多上传几张
required Boolean true 是否必传
picName String 'pName' 图片参数字段
map-表单地图
参数 数据类型 示例 说明
btns Boolean true 按钮组,绘制 重汇
btnToElm String 'areaType' btn 关联select name
btnToFilter String 'areaType' btn 关联select filter
btnToVal String '3' 用于触发行政围栏弹窗的val值
btnElmJson Object { 0:'circle', 1:'rectangle', 2:'polygon' } 绘制百度地图围栏的类型
admin Object { 'pointList':'pList', //传给后台的经纬度key（经纬度逗号相隔） 'circleRadius':'circleRadius', //若绘制圆，传给后台的圆心半径 'circleCenter':'circleCenter' //若绘制圆，传给后台的圆心坐标 } 行政围栏后台字段自定义
lf Number 300 左侧表单宽度 ,注:打开此弹窗宽度为必传字段(lf+rt)
rt Number 500 右侧地图宽度 ,注:打开此弹窗宽度为必传字段(lf+rt)
height Number 400 内容整体高度
以下是一个表单初始化的应用例子
var commonObj={
    id: 'yui-add',
    filter: 'yui-add',
    submitFn: function (data) {
        //data 表单数据对象 
    },
    callback:function(obj){
        //obj 当前表单对象
    },
    pcfg:{
        
        auto:false ,   //自动上传
        multiple:true, //多选
        url:app.url+ 'file/up',
        max:3,         //图片最大张数
        required:true, //是否必传
        picName:'pic1' //接口图片字段
    },
    verType:'tips', //提示方式 支持 msg tips alert
    item: [
        {label:'开始时间',name: "startTime",verify:'required', qt: 'gt',dt: 'date',
            cfg:{type: 'datetime',format:'yyyy-MM-dd HH:mm:ss'}
        },
        {label:'结束时间',name: "endTime",verify:'required',qt: 'lt',dt: 'date',
            cfg:{type: 'datetime',format:'yyyy-MM-dd HH:mm:ss'}
        },
        {label:'操作',name:'opt',qt:'eq',dt: 'switch',cfg:{values:'1|2',text:'男|女'}},
        {label:'部门',name:"deptId" ,verify:'required' ,qt:'eq',eventName:'tree',treeCfg:{
            url:'/page/default/tpl/tree.html?p=tree',
            title:'请选择部门',
            name:'name'
        }},
        {label:'性别',elmType:'select',name: "userSex",qt: 'eq',isBlock:true,verify:'required',cfg:{
            data:{'1':'男','2':'女','3':'保密','4':'你好'},count:0,values:'1',search:true
        }},
        {label:'食物',elmType:'select',name: "food",qt: 'eq',isBlock:true,verify:'required',cfg:{
            count:4,search:true,url:'select.js',cfg:{
                type:"get",
                searchName: 'foodName',         //自定义搜索内容的key值
                keyName: 'foodName',            //自定义返回数据中name的key, 默认 name
                keyVal: 'id',                   //自定义返回数据中value的key, 默认 value
                beforeSuccess:function(id, url, searchVal, result){
                    return result.data.data.data;
                }
            }
        }},
        {label:'地点',name:"address"/* ,verify:'required' */,qt:'eq',isBlock:true,eventName:'address',treeCfg:{
            url:'/page/default/tpl/address.html?p=address'
        }},
        {label: '描述',elmType: 'textarea',verify:'required',name: "remark",isBlock:true,qt:'like'}
    ]
},
popup=app.ui.autoform(commonObj);
2.表单重置popup.reset()
用于表单重置信息( 一般用于新增操作之前的表单重置 )
3.表单数据回显popup.set()
用于表单回显信息( 一般用于修改操作数据回显 ),该方法有5个形参，具体参数参见下表格
参数 数据类型 示例 说明
第一个参数 Object 无 选中行的数据
第二个参数 Array 无 待特殊回显的数据name的集合( 例如部门ID,角色ID等 )
第三个参数 Array ['lng','lat'] 待回显经纬度的后台字段集合( 暂只支持一个表单一个取点弹窗,多个数据回显尚未集成)
第四个参数 String 'picName' 待回显的图片后台字段(暂只支持一个表单一个图片字段,多个图片字段尚未集成))
第五个参数 Object { 'points':'pList', //数据回显点集合后台字段 'cirRadius':'circleRadius',//数据回半径后台字段 'cirCenter':'circleCenter',//数据回圆心后台字段 'admin' :true //用于区别行政围栏以及其他围栏 } 其他数据的回显
4.表单特殊数据回显popup.aliasSet()
用于表单回显特殊信息( 例如部门回显,一般用于修改操作特殊数据回显 ),该方法有2个形参，具体参数参见下表格
参数 数据类型 示例 说明
第一个参数 Object 无 选中行的数据
第二个参数 Object {'deptId':'deptName'} 待特殊回显的数据,key为要填充的表单元素的name,value为要填充进去的后台字段( 例如部门 )
5.表单弹窗打开popup.popup(para)
打开表单弹窗,该方法有一个形参，参数如下
参数 数据类型 示例 说明
title String '表单信息' 弹窗标题
btns Array [ {'title':'确定','yes':true}, {'title':'继续添加'，handle:function(){}}, {'title':'取消'} ] 弹窗按钮( 默认 确定，取消按钮),确定按钮必传yes参数,,,其他按钮可自定义事件，默认点击关闭
success Function 无 弹窗成功打开后的回调函数
width Number 500 弹窗宽度，默认500,最小300
height Number 300 弹窗高度，默认自适应
anim Number 1 layer内置弹窗动画,1-6,默认4
6.表单弹窗关闭popup.close()
关闭表单弹窗
7.表单元素禁用与取消禁用popup.disabled()
表单元素禁用与取消禁用(input/textarea)，该方法有2个形参，具体参数参见下表格
参数 数据类型 示例 说明
第一个参数 Array 无 待禁用的表单元素name的集合
第二个参数 Boolean true 禁用true ,取消禁用不传或者false select暂不支持
8.select下拉框禁用与取消禁用popup.selDisabled()
select下拉框禁用与取消禁用(input/textarea)，该方法有1个形参，具体参数参见下表格
参数 数据类型 示例 说明
第一个参数 Object popup.selDisabled({ 'dept':true //dept代表要禁用/取消禁用的表单元素name值 true 代表禁用,false代表启用 }) 待禁用的表单元素names
9.表单select联动popup.selLinkage(para)
表单select联动({})
参数 数据类型 示例 说明
clickSel String 无 操作的select id(ID为 当前弹窗ID_操作select Name)
linkSel String 无 联动的select id(ID为 当前弹窗ID_联动select Name)
url String 无 联动的select 服务器地址
count Number 1 区分联动select的多/单选,默认多选(0),可取值0,1
type String 无 联动的select 请求方式,默认post
cfg Object 无 联动的select 配置,参照select配置
keyName String 无 代表联动select根据操作select添加的额外字段名称，默认 id 
popup.selLinkage({
    clickSel:'yui-add_userSex',
    linkSel:'yui-add_food',
    cfg:{
        data:{
            name:'111' //额外参数
        },
        type:"get",
        searchName: 'foodName',         //自定义搜索内容的key值
        keyName: 'foodName',            //自定义返回数据中name的key, 默认 name
        keyVal: 'id',                   //自定义返回数据中value的key, 默认 value
        beforeSuccess:function(id, url, searchVal, result){
            return result.data.data.data;
        }
    },
    type:'',
    url:'select.js', //后台地址
    keyName:'ids' //
});
